//Error Message
error.appinit,응용 프로그램 시작 중에 에러가 발생했습니다.\n보다 자세한 정보를 얻기 위해서는 설정 내의 「예외 핸들러를 사용」를 OFF로 맞춰 주십시오.
error.apprun,실행 중에 내부 에러가 발생하여 응용 프로그램을 종료합니다.\n보다 자세한 정보를 얻기 위해서는 설정 내의 「예외 핸들러를 사용」를 OFF로 맞춰 주십시오.
error.cannotshowusagegraph,동작 시간이 부족하여 사용 시간 그래프를 표시할 수 없습니다.\n1분 뒤에 다시 시도해 주십시오.
error.invalidipport,지정한 포트 번호는 사용할 수 없습니다. 1024번~65535번 사이에서 정해 주십시오.
error.sstp.cannotlisten,지정한 포트 번호로 연결하는 데 실패했습니다.\n다른 응용 프로그램들과 포트 번호가 충돌하지 않는지, 혹은 포트 번호에 잘못된 값을 설정하지 않았는지 확인해 주십시오.
error.nofile,동작에 필요한 파일 또는 폴더가 없습니다.\n새롭게 설치하거나 수동으로 필요한 파일을 보충해 주십시오.
error.browserexec,브라우저 실행에 실패했습니다.\n설정->외부 앱->브라우저 설정을 확인해 주십시오.

error.tempboot,임시 파일용 폴더(Temp 폴더) 안에서 실행 중입니다.
error.uacboot,사용자 계정 컨트롤(UAC) 관리 하의 폴더에서 실행 중입니다.
error.asciiboot,문자 코드 문제로 파일에 접근할 수 없는 폴더에서 실행 중입니다.
error.alreadyrunning,SSP가 이미 실행 중인 것 같습니다.
error.bootanyway,이대로 사용하면 데이터가 손실될 수 있습니다. 그래도 실행하시겠습니까?
